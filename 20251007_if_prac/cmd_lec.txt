1. Чтобы запустить команду, надо набрать несколько слов и нажать Enter
2. Первое слово называется командной программы, остальные - аргументами командной строки
Например: py hello.py
Здесь программа-интерпретатор питон (py)
Её первый аргумент - путь к файлу нашей программы
3. Перемещение между папками происходит с помощью команды cd = change directory
cd .. переход на уровень выше (произносится цэдэ две точки)
cd 20251007_if_prac переход в указанную папку (директорию)
